# D&R Concrete and Restoration Website

A professional business website for D&R Concrete and Restoration, built with Next.js 15, TypeScript, and Tailwind CSS.

## Features

- **Modern Design**: Industrial, trustworthy design with custom brand colors
- **Responsive**: Fully responsive across all devices
- **Fast**: Built with Next.js 15 App Router for optimal performance
- **Accessible**: Semantic HTML and proper ARIA labels

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js App Router pages and layouts
- `/components` - Reusable React components
- `/public` - Static assets

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
- Google Fonts (Oswald, Roboto)

## Brand Colors

- Primary (Dark Slate): `#2F4F4F`
- Accent (Safety Yellow): `#FFD700`
- Background: `#F9FAFB`
